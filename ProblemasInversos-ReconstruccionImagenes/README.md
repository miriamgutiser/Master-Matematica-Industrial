# Problemas Inversos y Reconstrucción de Imágenes

Prácticas en MATLAB enfocadas en reconstrucción mediante técnicas de inversión numérica:

- Inversión algebraica de señales
- Reconstrucción basada en bases de datos
- Estudio del problema del calor hacia atrás
- Procesado numérico de señales e imágenes

Estas prácticas abordan la resolución de problemas mal planteados, típicos de imagen médica y tomografía.

**Lenguaje principal:** MATLAB
